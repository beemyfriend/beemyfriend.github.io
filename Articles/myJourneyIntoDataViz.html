I got a job

Purpose: I'm no longer just a fan. But now I realize how large and really diverse the field really is.

  • I got a job in data visualization as a data viz engineer
  • I didn't really know what to expect, in hindsight, saying I wanted to work in data viz is as vague as saying I wanted to work in data science
  • Data viz is everywhere, because it is everywhere, there is so much to learn and do
  • My only recommendation for someone trying to get into data visualization, is narrow it down.
  • Carreers in data viz can be found in newsrooms, research labs, consulting firms, banks, cartography companies.
  • If you are interested in a particular field, there is almost definitely a need for data viz in that field
  • Everyone found their own unique path to the field of data viz. I've created a small write up trying to summarize how people 'do data viz'
